重點更新：
1. 首頁 UI 重整：主視覺 + 3 個主要入口 + 產品卡片 + FAQ。
2. 漢堡選單優化：右側滑出、背景遮罩、鎖定捲動、避免與 LINE 浮動鈕互蓋。
3. 產品頁改為動態架構：用 product.html + products.json，可持續擴充。
4. 內容頁補齊：龜鹿系列、食用方式、料理搭配、柒玄茶、FAQ、聯絡頁。
5. SEO 基礎補強：title、description、canonical、OG、JSON-LD、robots、sitemap。
6. 頁尾與聯絡方式更聚焦：主站以 LINE 為主要聯絡入口。

部署方式：
將本包檔案覆蓋到 GitHub Pages repo 根目錄即可。
images 資料夾仍沿用你 repo 既有圖片，不需另外替換。
